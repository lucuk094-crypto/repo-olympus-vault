🚀 CARA MENJALANKAN:

1. Install Dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Jalankan Local:
   ```bash
   python backend/app.py
   ```

3. Atau Deploy ke Railway (RECOMMENDED):
   ```bash
   npm install -g @railway/cli
   railway login
   railway init
   railway up
   ```
