Generate SECRET_KEY:
python -c "import secrets; print(secrets.token_hex(32))"

⚠️ IMPORTANT NOTES:

Vercel & Netlify:
❌ Ephemeral storage - file akan hilang saat restart
✅ Cocok untuk: Demo, testing, preview
❌ Tidak cocok untuk: Production dengan file upload

Railway (Alternative):
✅ Persistent storage - file tidak hilang
✅ Gratis untuk mulai
✅ Cocok untuk production

Deploy ke Railway:
```bash
npm install -g @railway/cli
railway login
railway init
railway up
```
